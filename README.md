📱 To Do App (Flutter + Hive Database)

A simple and efficient To Do Application built using Flutter, designed to help users manage daily tasks with ease. The app uses Hive, a lightweight and fast local database, to ensure smooth performance and reliable offline storage.

✨ Features
📝 Create Tasks – Add new to-do items quickly and easily
👀 Read Tasks – View all saved tasks in a clean and organized list
✏️ Update Tasks – Edit existing tasks anytime
🗑️ Delete Tasks – Remove completed or unnecessary tasks
💾 Offline Storage – All data is saved locally using Hive, so the app works without internet connection
⚡ Fast Performance – Lightweight local database ensures quick read/write operations

🧠 Key Highlights
- Built with Flutter (Dart) for cross-platform support
- Uses Hive database for efficient local storage
- Implements full CRUD (Create, Read, Update, Delete) functionality
- Fully offline-first application, no internet required
- Simple and clean UI for better user experience

🎯 Purpose
This project demonstrates how to build a functional offline-first mobile application using Flutter and Hive, focusing on local data persistence and CRUD operations.
